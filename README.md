## 🧩 Week-0: Scratchcard Challenge

This assignment consists of two programming tasks based on a scratchcard puzzle.

### 📄 Part-1
- Reads scratchcard data from `input.txt`
- Counts matching numbers
- Computes the score for each card
- Generates the final score in `output.txt`

### 📄 Part-2
- Simulates the scratchcard copy mechanism
- Tracks additional cards won through matching numbers
- Computes the total number of scratchcards obtained
